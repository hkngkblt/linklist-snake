LinkList Snake Game v1.0

Builded with C & SDL2 Libraries
Works Only With 64-Bit Machines
Tested on Ubuntu 19.10 & Windows 10 1909 & 1809 versions.

-UP DOWN LEFT AND RIGHT TO MOVE
-ESC TO EXIT

--- FOR LINUX ---
-Install libSDL2 & libSDL2_mixer to your system using package manager like apt.
-Set linker settings on your C IDE.
-Compile & RUN.
